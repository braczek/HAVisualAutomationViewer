## Installation via HACS

1. Add this repository to HACS as a custom repository
2. Search for "Visual AutoView" in HACS
3. Install the integration
4. Restart Home Assistant
5. Add the integration through the Home Assistant UI

## Manual Installation

1. Copy the `custom_components/visualautoview` directory to your Home Assistant `custom_components` folder
2. Restart Home Assistant
3. Configure the integration

## Usage

After installation, access the Visual AutoView interface to:
- View automation graphs
- Search automations by name, entity, or configuration
- Analyze performance metrics
- Explore entity relationships
- Export automation visualizations

For detailed documentation, visit the [GitHub repository](https://github.com/braczek/HAVisualAutomationViewer).
