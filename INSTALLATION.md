# Installation Guide for Visual AutoView

## Installation via HACS (Recommended)

1. Open Home Assistant
2. Go to **Settings** → **Devices & Services** → **Custom Integrations** (or use the HACS menu)
3. Click **"Create Automation"** or search for **Visual AutoView**
4. Follow the installation prompts
5. Restart Home Assistant

## Manual Installation

### Using PowerShell (Windows)
```powershell
.\install.ps1
```

### Using Bash (Linux/macOS)
```bash
bash install.sh
```

## After Installation

1. Restart Home Assistant
2. Clear your browser cache
3. Access the Visual AutoView panel from the sidebar

## Troubleshooting

If you encounter issues:
- Check the Home Assistant logs for errors
- Ensure your Home Assistant version is 2023.11.0 or later
- Clear the browser cache and reload the page
- Check that the `custom_components/visualautoview` folder exists
